# Contributing

## Standards
- C99.
- No dependencies.
- Mandatory input validation via `is_safe_name` and `is_safe_path`.
- No comments.

## Roadmap
- Parallel status check (pthreads).
- SQLite registry backend.
- Command aliases.

## Good First Issues
- Add language preset in `src/languages.c`.
- Implement version flag.
- Improve templates in `include/templates.h`.
